# Introduction

This style guide acts as a home for Canopy brand guidelines, styling guidance and
helpers, and UI components. Sapling creators are encouraged to use these
suggestions and tools while building their Saplings to create a smooth user
experience.

<br />
<br />

## Motivation

<ColumnBlock id="motivations">
  <ColumnItem>
    <b>Cohesive design</b>
    <p>
      The Canopy design system provides Sapling developers with the tools to
      create Saplings that blend seamlessly with the Canopy interface and
      provide users with a delightful experience.
    </p>
  </ColumnItem>
  <ColumnItem>
    <b>Ease of use</b>
    <p>
      The design system provides easy-to-use classes and components to build and
      style your Saplings.
    </p>
  </ColumnItem>
</ColumnBlock>
